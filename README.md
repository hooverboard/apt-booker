# Apt-Booker: Appointment Booking System

Apt-Booker is a full-stack web application that allows users to book appointments with various shops and enables shop owners to manage their services and bookings.

## Features

* **User Authentication:** Secure user registration and login.
* **Shop Management:** Shop owners (hosts) can create, update, and delete their shops.
* **Service Management:** Hosts can add, edit, and remove services for their shops, including details like price and duration.
* **Appointment Booking:** Users can browse shops and book appointments for available time slots.
* **Appointment Viewing:** * Users can view their upcoming and past appointments.
    * Hosts can view and manage all appointments for their shops.
* **Dynamic Availability:** The system calculates and shows available appointment times based on shop hours and existing bookings.
